upscale markets

Online grocery store

Responsive prototype for an online grocery store web app that offers a rewards system, pick-up and delivery.

Features

  • Add items to cart
  • Edit profile
  • View last order details
  • View cart items
  • View point redemption items

Development

Upscale Markets is built with HTML, CSS and JavaScript (AngularJS), using Google's Angular Material framework. It is co-developed with Matthew Tsui and Brian Mouallem.
